# Comprehensive-Analysis-of-Pizza-Place-Sales-Data

Pizza Place Sales Analysis – Project Summary

In this task, I analyzed a year’s worth of sales data from a pizza place. The dataset contained four related tables: Orders, Order Details, Pizzas, and Pizza Types, which I merged into a single dataset for analysis.

I calculated key business metrics such as total revenue, total quantity sold, total number of orders, number of pizza types, and average pizza price. I also analyzed peak sales hours, daily and monthly sales trends, top 5 best-selling pizzas, and identified underperforming menu items.

This task helped me practice data cleaning, dataset merging, exploratory data analysis (EDA), and generating business insights from real-world sales data.
